# `bin_to_ASCII`
비트씩 끊어서 들어오는 입력을 ASCII로 변환하여 출력하는 프로그램

- 입력: N개의 8비트 덩어리
- 출력: 이어진 ASCII문자

ASCII 테이블이 있어야 한다.
- `char` 타입의 배열이다.

1. 입력을 받아 몇 바이트가 들어왔는지 파악한다.
2. 메모리를 동적 할당하여 입력된 바이트들을 문자열 덩어리로써 저장한다.
3. 각 바이트에 대해 다음을 반복한다.
	1. 2진수 문자열을 정수로 변환한다.
	2. 변환된 정수를 ASCII 테이블의 인덱스로 사용해 문자를 출력한다.

# 사용 방법
---
1. 프로그램을 컴파일한다.
```shell
gcc bin_to_ASCII.c -o bin_to_ASCII
```
2. 인자로 비트열을 바이트 단위로 전달한다.
```shell
./bin_to_ASCII 01001000 01100101 01101100 01101100 01101111 00101100 00100000 01000100 01110010 01100101 01100001 01101101 01101000 01100001 01100011 01101011 00100001
```
3. 출력 결과를 확인한다.
```shell
Hello, Dreamhack!
```
